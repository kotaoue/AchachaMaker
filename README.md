# AchachaMaker

A video maker made by me, for me.

## Getting Started (macOS)

### Prerequisites

- [uv](https://docs.astral.sh/uv/)
- ffmpeg
- (Optional) [VOICEVOX](https://voicevox.hiroshiba.jp/) desktop app running locally for voice synthesis

### Setup

```bash
uv sync
```

### Run Tests

```bash
uv run pytest

uv run python main.py
```
